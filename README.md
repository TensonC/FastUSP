# FastUSP USB转串口通信工具
Fast USB to Serial Port

简易的串口通信工具，可接收发送，配置串口信息，支持部分ANSI转义码

ncurses库竟然不支持ANSI转义字符，功能是自己简单实现的，因此不太好用，支持的转义字符也极少(-^-")
目前逐渐对该项目失去兴趣(' - ')，准备学其它东西了,因此不想再花时间添加功能或调试ANSI转义部分啦，
如果未来有机会或有必要的应用场景，会再尝试完善(大概
不过接收和发送信息都是能用的。

## 执行
可以使用 fusp build 此shell文件进行编译，也可以直接使用fusp可执行程序

## 使用
示例`./fusp open 0 -b 115200 -c N81`
表示打开设备列表中的设备0,波特率 115200,无校验位(N)，字节数 8,停止位 1

### 可用命令:
- list 查看设备列表
- open num 打开设备列表中设备号为 device#num 的设备
- help 查看帮助
- version 查看版本信息

### 可用参数:
- -b 设置波特率
- -c 设置串口信息
- -n 设置打开串口终端名称(暂时没什么用)
- -v 查看版本信息
- -h 查看帮助

## 界面
打开设备后，主要界面展示接收到的信息，右侧显示该设备路径和串口的配置信息
底部一行发送窗口，输入要发送的内容后，按下回车键即可发送，会自动在发送信息的最末尾添加一个回车，
即 `\n`
